# OpenPlayer v1.5.1

## Changes

- Refines plugin side-panel presentation so custom views align with the window
  controls and transport bar, keep rounded clipping, and dismiss when users
  click back on the playback surface.
- Adds setting-backed side-panel iframe opacity so plugins can expose
  user-tunable transparency while still blending over the native mpv video
  surface on Windows WebView2.
- Updates custom plugin views to receive the effective user accent override and
  documents semi-transparent themed side-panel surfaces for SDK authors.
- Adds the `tv` plugin action icon for IPTV and other TV-like channel browser
  plugins.
- Refreshes the official IPTV Channel Switcher plugin with a translucent
  theme-driven panel and TV-specific action icon.
- Keeps official plugin packages aligned with the 1.5.1 host and SDK contract.

- Windows NSIS installer
- Linux `.deb` package
- Linux AppImage
