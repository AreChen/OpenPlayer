# OpenPlayer v1.6.3

## Changes

- Fixes a Windows resize hang caused by the overlay window feeding its own
  resize events back into the main video window.
- Keeps the macOS-only overlay-driven resize path isolated to macOS, where the
  overlay is intentionally user-resizable.
- Adds a shell architecture check so overlay resize events cannot regress into
  driving the main video window on Windows.

## Packages

- Windows NSIS installer
- Linux `.deb` package
- Linux AppImage
- unsigned macOS DMG
- SHA256 checksum files
