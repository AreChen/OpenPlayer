# OpenPlayer v1.6.2

## Changes

- Restores smooth native resize behavior on macOS by letting the overlay window
  own the platform resize interaction and synchronizing the main video window.
- Keeps fullscreen, window movement, and overlay alignment routed through the
  existing two-window mpv host architecture.
- Removes the temporary in-app resize boundary indicator now that native macOS
  resize behavior is working reliably.
- Adds release checks to prevent the temporary resize feedback layer from being
  reintroduced.

## Packages

- Windows NSIS installer
- Linux `.deb` package
- Linux AppImage
- unsigned macOS DMG
- SHA256 checksum files
