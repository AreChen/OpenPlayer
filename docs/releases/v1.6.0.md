# OpenPlayer v1.6.0

## Changes

- Upgrades the plugin SDK contract to 1.6.0 with broader module-style host
  APIs for media segment export, playlist updates, recording state, plugin
  storage scans, and reusable custom-view UI primitives.
- Adds host-controlled media segment export so plugins can export video or audio
  clips with validated paths, formats, timestamps, and overwrite behavior.
- Adds the official Segment Export plugin for choosing clip ranges and exporting
  video or audio from the current media.
- Extends plugin runtime storage with prefix-limited listing plus byte-usage and
  per-value limit metadata for safer plugin-owned redb data.
- Extends custom plugin views with a theme-aware `.op-*` UI kit for buttons,
  forms, switches, sliders, tables, keyboard tokens, and compact panels.
- Fixes the Windows Alt+F4 path so closing the native mpv video host also closes
  the overlay shell instead of leaving a black UI window.
- Refreshes desktop dependencies, Rust tooling, Tauri crates, Vite, and the
  bundled Windows LGPL mpv runtime.
- Keeps official plugin packages aligned with the 1.6.0 host and SDK contract.
