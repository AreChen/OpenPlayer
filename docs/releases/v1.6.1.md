# OpenPlayer v1.6.1

## Changes

- Fixes a macOS shutdown hang when closing OpenPlayer while libmpv playback is
  active from the custom window controls.
- Coordinates main video window and overlay close requests with a shared guard
  so native close events cannot recursively close both windows.
- Tears down the embedded mpv player before closing windows, keeping the macOS
  AppKit video host cleanup on the main thread when required.
- Adds release checks to keep the close-teardown path from regressing.

## Packages

- Windows NSIS installer
- Linux `.deb` package
- Linux AppImage
- unsigned macOS DMG
- SHA256 checksum files
